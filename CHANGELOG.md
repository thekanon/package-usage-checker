# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-26

### Added
- Initial release
- Package list file parsing
- Package installation detection from package-lock.json
- Source code scanning for package usage
- Support for ES6 imports and CommonJS requires
- Support for scoped packages (@scope/package)
- Detailed usage report generation
- Command-line interface with customizable options

### Features
- ✅ Detect installed packages
- ✅ Detect used packages in source code
- ✅ Identify unused dependencies
- ✅ Generate comprehensive reports
- ✅ Support multiple import patterns
